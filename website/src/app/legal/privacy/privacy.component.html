Learn what data is collected and how it is used.

Please be aware that this is only an open source front-end software. ALL the data associated with a shop is saved on the blockchain and read out by interacting with the individual contracts. The creator of this shop software has no influence over who is reading this public data.
The shop frontend is hosted via IPFS and thus can be decentrally hosted by multiple persons or organizations around the globe.
If you are concerned that a IPFS gateway could gather user data, please run a own gateway, or just download the code of this shop and run it locally on your machine.

When a user creates his shop, all associated data is put into Arweave via the Bundlr network. Arweave is a storage centric blockchain and Bundlr operates a bridge to allow data to be saved via Ethereum. Please check with Bundlr if they collect personal information like the IP address when you interact with their bridge.

For encrypting and decrypting content files you interact with the Lit Protocol. Please check on their website if and how they might collect personal data.

Temporary data is put into the local storage of your browser to optimize the user experience. This is data that is not personally related. Its data like certain IDs to pickup aborted processes during normal operation of the shop software.

When you create a shop you mint an NFT that is originally associated with your wallet address that was used create your shop. If you want to display all the data that is saved in the shop contract, please use the link in the footer of this page.
