Why is a wallet required?

From a user perspective, it would be nice to display the shop without any connected wallet. Howver important information about the shops initial data is inside a Smart Contract and must be read to properly display the shops content.
A wallet is used to query this information from the chain initially to bootstrap the discovery process.

Which Wallet?

One of the mostly used wallets is the Metamask Browser extension. But the shop is also compatible with wallets that support Wallet Connect. Just choose this option when you connect your wallet if you want to use your mobile phone wallet for example.
